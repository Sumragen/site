# Api endpoints documentation

# Routes

## User

### check user in the array

```js
    {
        url:'/signIn',
        method:'POST',
        data:'user'
    }
```

### get current user

```js
    {
        url:'/getUser',
        method:'GET',
        data:'currentUser'
    }
```

### add user to array of users

```js
    {
        url:'/users',
        method:'POST',
        data:'user'
    }
```
