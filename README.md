# Api endpoints documentation

# Routes

## User

### checks whether the user is registered

```js
    {
        url:'/signIn',
        method:'POST',
        data:'checkUser'
    }
```

### Get list of events

```js
    {
        url:'/events',
        method:'POST',
        data:''
    }
```

### Get schedule

```js
    {
        url:'/schedule',
        method:'POST',
        data:''
    }
```

### add user to array of users in datasource

```js
    {
        url:'/register',
        method:'POST',
        data:'user'
    }
```
